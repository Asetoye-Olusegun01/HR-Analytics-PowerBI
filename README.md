# HR Analytics Dashboard (Power BI)

This project presents an interactive HR Analytics Dashboard built in Power BI to help organizations visualize and understand employee demographics, performance, and retention trends. The dashboard transforms raw HR data into actionable insights that support strategic HR decision-making.

## Key Insights
- Workforce distribution varies significantly across departments and job roles.
- Certain age groups and departments experience higher attrition, indicating retention challenges.
- Performance ratings show clear patterns across job roles and years of experience.
- Gender representation is uneven in some departments.
- Employees with lower tenure tend to have higher turnover risk.

## Objectives of the Dashboard
- Analyze employee demographics (age, gender, education).
- Understand job role and department distribution.
- Evaluate performance ratings across the organization.
- Identify turnover patterns and possible retention issues.
- Support HR teams with data-driven insights.

## Dataset Description
The dataset includes the following fields:
- Employee ID  
- Full Name  
- Age  
- Gender  
- Department  
- Job Role  
- Education Level  
- Hire Date  
- Years of Experience  
- Performance Rating  
- Attrition/Turnover Status  

Data cleaning and transformation were done using Power Query, and additional KPIs were created using DAX.

## Tools Used
- Power BI Desktop  
- Power Query Editor  
- DAX  
- Microsoft Excel
